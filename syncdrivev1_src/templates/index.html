Guide

How to use SyncDrive

Getting Started

  1. Select a directory - Click the sessions path in the sidebar
  2. Review sessions - Check the Sessions page
  3. Add to queue - Click "+ Add to Queue"
  4. View results - Check the Processed page for visualizations

Session Structure

session_name/
  ├── melb-01-cam-01/    # Camera folders
  │   ├── seg_001.mp4
  │   └── ...
  ├── can_raw.csv        # CAN bus data
  ├── phone/             # Phone sensors
  └── watch/             # Watch sensors

Data Sources

  • Cameras - Video recordings concatenated into single files
  • CAN Bus - Vehicle data parsed into human-readable CSVs
  • Phone/Watch - Sensor data (accelerometer, gyroscope, GPS)

Processing Output

  • *_full.mp4 - Concatenated videos
  • parsed/ - CAN data (wheel_speed.csv, brake.csv, etc.)
  • metadata.json - Processing info with time ranges
